Annotation

Do you want to learn how to program? Try the new Java textbook for high school students and their teachers. The explanation is demonstrated with examples and output to a web browser. In addition to the programming language, the book also explains concepts from object-oriented programming, regular expressions, UML class diagrams, web concepts, Maven projects, and some other related topics. The text is supplemented with color illustrations, charts, and real screen shots. Java has long been one of the most in-demand programming languages in the information technology job market. A sample of the book is here.

 

For whom the book is intended

The book is aimed at high school students, but the breadth of the topic and the concept of the examples included may appeal to educators and other readers interested in information technology and the development of Java-based software applications.

 

Why I wrote the book

When I started with Java, I wondered if it was possible to write a textbook in a more attractive way. After years of practice, I published a short article about building web pages in Java, based on the object model of HTML elements, and while doing so, I thought this might be a great vehicle for creating interactive textbook examples while learning the language. At the nearest bookstore, I opened one of the new textbooks on the language and saw, once again, a formal description with black and white pictures, academic examples and a few graphical examples based on the now defunct Swing library. That's when I decided to embark on the uncertain path of an innovative programming textbook writer.

About author

Pavel Ponec (born in 1965) graduated from Brno University of Technology in automation and since then he has been working as a software developer and analyst, since 2000 he has been working mainly in Java with web technologies. In 2008 he received the Certified Programmer for the Java 2 Platform, Standard Edition 5.0 certificate from Sun Microsystems.

Publicly available reference projects
