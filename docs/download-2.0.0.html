Příklady

Po zakoupení knihy si prosím uschovejte doklad o nákupu, protože vás opravňuje používat ukázkové příklady k soukromým i komerčním účelům. Distribuce zdrojového kódu k této knize je možná pouze vlastníkem knihy při zachování původního komentáře v hlavičce. Třetí strana je oprávněna použít zdrojové kódu pouze pro své vlastní studium, jinak je třeba získat vlastní licenci zakoupením knihy nebo e-booku. V případě nejasností se obracejte na autora knihy, kontakt je zde. Zvažte prosím, že výroba knihy není jedinou nákladovou položkou, další výdaje souvisí s propagací a distribucí. Děkuji tímto za podporu i pochopení. Stažením souboru vyjadřujete souhlas s těmito podmínkami. Příklady jsou k nahlédnutí zde, a odkaz ke stažení zdrojových kódů je tady.

 

Anotace

Průvodce začátečníka programovacím jazykem Java od jeho základů až po interaktivní webové aplikace. V první části knihy čtenář získá znalost pojmů objektového programování, přehled o datových typech a užitečných metodách standardní knihovny jazyka, setká se s pojmy potřebnými pro stavbu webových stránek. Druhá část knihy přináší výklad použitých programových konstrukcí na pozadí řešených aplikací. Java patří dlouhodobě mezi nejžádanější programovací jazyky na trhu práce v oboru informačních technologií. Ukázka knihy je tady.
Doporučená Java k příkladům je ke stažení zde.
